# Training follow-up email

**Subject:** Following up from last week's BaronsHub session

Hi all,

Thanks so much for joining last week's introduction to BaronsHub, and a huge thank you to Tanya for getting the session set up so smoothly.

Since we wrapped up I've been working through the list of enhancements we talked about on the day, and I'm pleased to say they're all now live in BaronsHub for you to start using:

**Planning and events**
- You can attach an event or planning item to more than one venue in one go, rather than copying it out multiple times.
- "Propose an event" now sits in your sidebar. Anyone can put an event forward and it lands in a queue for an admin to approve.
- Every planning item has its own page now, with a full history of changes and space to upload attachments.
- The calendar shows everything — past as well as upcoming — and you can toggle completed or cancelled items in and out.

**Tasks and SOPs**
- You can add quick notes against any task. Just type and press Enter to save.
- You can mark a task as "not required" when it genuinely doesn't apply, rather than leaving it sitting there open.
- SOP tasks can be tailored per venue, so each pub gets the steps that make sense for them.
- You can upload files against tasks and events — menus, artwork, sign-off sheets, anything useful.

**Debriefs and leadership**
- Labour hours can be captured on the debrief.
- When a debrief is submitted the leadership team get an automatic email summary.

**Day-to-day things that should make life easier**
- The dashboard has been rebuilt around a single todo list, so you can see at a glance what's on your plate.
- If a customer tries to book twice, you'll be prompted to update their existing booking rather than create a duplicate.
- Manager Responsible on events and venues now picks from the user list instead of a free text box, so no more typos or missed handovers.
- User management has been tidied up so accounts can be deactivated or removed cleanly.
- You should stay logged in between visits now — no more surprise sign-outs.
