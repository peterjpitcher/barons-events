## Middleware

- **File**: `middleware.ts` (project root)
- **Matcher**: All routes except `/api/*`, `_next/static`, `_next/image`, `favicon.ico`
- **Auth flow**: Supabase JWT (`getUser()`) + custom app-session validation (`validateSession`)
- **Public paths**: `/login`, `/forgot-password`, `/reset-password`, `/auth/confirm`, `/unauthorized`, `/deactivated`, `/l/*`
- **Special handling**: Domain rewrite for `l.baronspubs.com` (event landing pages + short links)
- **Security**: CSP with nonce, CSRF token, HSTS, X-Frame-Options DENY, deactivation check

See [[overview#Middleware Details]] for full breakdown.
