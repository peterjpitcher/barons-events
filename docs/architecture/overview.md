## Domain

BaronsHub is an event management platform for the Barons hospitality brand. Core capabilities:

- **Event lifecycle**: draft -> proposed -> approved/rejected -> published -> completed
- **Multi-venue**: events can span multiple venues; office workers are venue-scoped
- **Booking management**: customer bookings with SMS campaigns and reminders
- **Planning**: recurring planning items, series, tasks with venue assignments
- **SOP templates**: standard operating procedure checklists auto-generated per event
- **Debriefs**: post-event feedback and review
- **Public API**: read-only REST API for website integration (v1)
- **Short links**: branded short URLs with click tracking
